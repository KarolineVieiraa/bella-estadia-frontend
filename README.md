# Bella Estadia
Criamos o Bella Estadia com o objetivo de ser um sistema de hotelaria mais acessível, tendo como público alvo pequenos empreendedores

## Tecnologias
Esse projeto é apenas ao front-end da aplicação web de hotelaria.

- React
- TypeScript
- Node.js
